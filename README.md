# Needleforge
A utility for adding Crests and Tools to the game (Crest and Tool mechanics not included)

## Features
- Add Crests
    - Create CrestData
    - Hooks for Injecting Code into the BindFSM
    - Custom HP Background
- Add Tools
  - Yellow and Blue are fully supported
    - Red Tools are WIP
      - Liquid Tools Are Working (Plasmium Phial/Flea Brew)
      - Projectile Tools are NOT working
- Add Custom Tool/Slot Colors
    - Pink (Red/White), Green (Blue/Yellow), and Black (All Colors) are pre included.

## Known Issues
- Custom Colored Tools Will Auto Equip no Matter What
